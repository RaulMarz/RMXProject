# FEMES - Fast Emergengy Support 

Xamarin Project

Aplicativo móvil de soporte para la atención de emergencias para el Cuerpo de Bomberos Voluntarios de Bogotá

Este aplicativo móvil apoyará el registro inmediato de información sobre los incidentes atendidos por el Cuerpo de Bomberos Voluntarios de Bogotá, sistematizando la información para posterior análisis que será enfocado a toma de decisiones, mejorando la gestión de unidades que atendienden las emergencias.

Proyecto de distribución privada
